Self guided collision avoidance car -- Goal

The goal of this project is to build a car that can successfully navigate a maze while using ultasonic sensor for collision avoidance. This was all done using the PYNQ-Z1 board.

Overview of steps taken to achieve our goal

With so much entailed in a self navigating car, at first we didnt know where to start. The first thing we did was make a list of objectives that must be completed to accomplish the task.

  • - Modify microblaze code to make our Ultrasonic Sensor functional
  • - Make sure all of the other peripherals are working, including H-Bridge, Motors, USB-Wifi, external Battery
  • - Assemble the car kit and connect all peripherals together
  • - Devolope an algorithm for avoiding walls
  • - Develope an algorithm to intelligently solve a maze
